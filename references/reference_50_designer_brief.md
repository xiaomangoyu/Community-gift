# 50 张 Reference 设计交接表

这份表给设计师使用。目标不是做 50 张完全不同的周边，而是做 50 张可复用的“应援棒视觉母型”。每张图都需要满足同一个基础产品规范：

- 黑色棚拍背景，1:1 正方形图。
- 固定 45 度朝右产品视角。
- 完整展示整支打 call 棒：灯头、连接处、握柄、底盖都要清楚。
- 灯头是视觉重心，中央有实体发光核心，文字嵌入核心内。
- 不要真人、人脸、外接文字牌、海报卡片、复杂场景。
- 测试文字就是画面里唯一主文字，尽量清晰、居中、有 3D 厚度。

## 给设计师的判断标准

一张 reference 合格的最低标准：

1. 一眼能看出它属于表里的 reference 类型。
2. 不是一堆元素贴上去，而是一个一体成型的收藏级应援棒。
3. 文字在灯头核心内，只有一处主文字。
4. 手柄和底盖也延续主题，不是默认白管。
5. 纯黑背景，完整棒体，不能裁切底部。
6. 外扩装饰可以丰富，但不能遮挡中央文字。

入库时每张图需要同时交付：

| 交付物 | 要求 |
|---|---|
| 图片 | 1:1，黑底，完整应援棒 |
| prompt | 中文完整 prompt，风格接近现有 `references/imgs/*.txt` |
| 测试文字 | 使用表里的想象主播文字 |
| 建议 tags | shape / color / material / vibe / text 五类 |
| 备注 | 这张图最适合覆盖哪些主播场景 |
