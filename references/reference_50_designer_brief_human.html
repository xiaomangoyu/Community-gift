50 张 Reference 设计交接表

目标是制作 50 张可复用的应援棒视觉母型。每张图都要覆盖一个真实主播/场景缺口，并且后续可以通过 tags 进入 reference manifest。

黑色棚拍背景，1:1 正方形，固定 45 度朝右产品视角。
完整展示整支打 call 棒：灯头、连接处、握柄、底盖都要清楚。
文字只在灯头中央核心出现，不能做外接文字牌或海报。
灯头是视觉重心，但手柄和底盖也要延续主题。
不是元素贴图，要像一体成型的收藏级实体产品。
外扩装饰可以丰富，但不能遮挡中央文字。
